# Express Authentication

User authentication with express. Tutorial from Stephen Grider on [Udemy](https://www.udemy.com/course/react-redux-tutorial).

### Installing

Clone the project
```
git clone <project url>
```

Install dependencies
```
npm install
```

Run app 
```
npm run dev
```

## Built With

* ExpressJS
* NodeJS
* MongoDB

## Features

* User Authentication
